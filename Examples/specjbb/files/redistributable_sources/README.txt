Information about SPECjbb2015 redistributable_sources directory
------------------------------------------------------------------------------------------------------------------------

The redistributable_sources subdirectory in the SPECjbb2015 Kit contains copies of the licenses 
that cover the freely-available and redistributable packages used in the SPECjbb2015 suite. 
License files are stored under the corresponding redistributable_sources/<package_name> subdirectories.

For some packages redistributable_sources/<package_name> directory also holds the package sources.
Rest of the sources may be found from the Source link.
