Thank you for selecting the SPECjbb2015 benchmark, advancing many new
features over the previous generation SPECjbb2005.

Before you begin deploying this benchmark, please consult the
SPECjbb2015 benchmark User Guide.

It is strongly recommended that you read the entire SPECjbb2015
benchmark Run and Reporting Rules before using this benchmark.

For an overview of the benchmark architecture, please refer the
SPECjbb2015 benchmark Design Document at the website:
http://www.spec.org/jbb2015/

For information on third-party open-source software incorporated in 
this benchmark, please refer to the redistibutable_sources
subdirectory.

EXECUTION:

A JRE (Java Runtime Environment), version SE 7 or later is required
for the execution of this benchmark. If using version SE 9, please
see the FAQ at http://spec.org/jbb2015/docs/faq.html

For any new updates, visit http://www.spec.org/jbb2015/

Happy Benchmarking!
